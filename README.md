# VaricoseVeinsDetectionAI
Varicose Veins Detection System is an AI-powered healthcare application using the AI/ML and trained on the CNN models.

=> Problem Definition
Varicose veins are enlarged and twisted veins that usually occur in the legs. Early detection is important because delayed diagnosis can lead to pain, swelling, skin ulcers, and circulation problems. 
Traditional diagnosis requires clinical examination and specialist consultation, which may not always be quickly accessible.

=> Objective
The objective of this project is to develop an AI-powered web application that can detect varicose veins from leg images using deep learning techniques. The system also provides:
Patient and doctor dashboards
Scan history management
Automated prediction reports
Confidence score generation
Medical record monitoring
The system aims to assist doctors and provide faster preliminary screening for patients.
